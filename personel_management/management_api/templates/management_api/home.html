API Documentation

Endpoint Method Description
/api/military/ GET, POST List all military personnel or create new entry
/api/military/{id}/ GET, PUT, DELETE Retrieve, update or delete specific military personnel
/api/military/{id}/children/ GET List children of specific military personnel
/api/military/{id}/diplomas/ GET List diplomas of specific military personnel
/api/children/ GET, POST List all children or create new entry
/api/diplomas/ GET, POST List all diplomas or create new entry

Filtering and Search Options

All endpoints support the following query parameters:

  • search: Search across specified fields
  • ordering: Order results by specified fields

Military Personnel Filters

  • rank: Filter by military rank
  • current_unit: Filter by current unit
  • first_name: Filter by first name
  • last_name: Filter by last name
  • date_of_birth: Filter by birth date

Children Filters

  • gender: Filter by gender (M/F)
  • school: Filter by school
  • grade: Filter by grade
  • date_of_birth: Filter by birth date

Diploma Filters

  • institution: Filter by institution
  • degree: Filter by degree
  • field_of_study: Filter by field of study
  • graduation_date: Filter by graduation date

Example Usage

/api/military/?rank=CPT&date_of_birth=1990-01-01&ordering=-service_number
/api/children/?gender=M&school=High School&search=John
/api/diplomas/?institution=University&ordering=-graduation_date
